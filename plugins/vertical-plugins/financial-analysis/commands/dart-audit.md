Load the `dart-financials` skill and execute the following workflow:

## Step 1: 기업 코드 조회
- DART MCP `resolve_corp_code`로 기업명 → corp_code 변환
- 동명이인 기업이 여러 개면 목록 보여주고 선택 요청

## Step 2: 감사보고서 수집
- `search_disclosures`로 해당 기업의 감사보고서(A001) 목록 조회
- 요청 기간 내 연도별 감사보고서 전부 수집
- `download_document`로 각 연도 감사보고서 원문 가져오기
- **비상장사도 외부감사 대상이면 DART에 감사보고서 존재** → 동일하게 처리

## Step 3: 재무제표 파싱
감사보고서에서 다음 항목 추출:

**손익계산서 (P&L)**
- 매출액 / 매출원가 / 매출총이익
- 판매비와관리비 (세부항목: 인건비, 감가상각비, 무형자산상각비 등)
- 영업이익
- 영업외손익 (이자수익, 이자비용, 기타)
- 세전이익 / 법인세 / 당기순이익
- EBITDA = 영업이익 + 감가상각비 + 무형자산상각비

**재무상태표 (BS)**
- 유동자산 (현금, 매출채권, 재고자산, 기타)
- 비유동자산 (유형자산 총계, 무형자산 총계, 투자자산, 기타)
- 유형자산 상세: 취득원가 / 감가상각누계액 / 장부가액
- 무형자산 상세: 취득원가 / 상각누계액 / 장부가액
- 유동부채 / 비유동부채 (차입금 구분)
- 자본금 / 자본잉여금 / 이익잉여금 / 총자본

**현금흐름표 (CF)**
- 영업활동현금흐름
- 투자활동현금흐름 (CAPEX 별도 표시)
- 재무활동현금흐름 (차입/상환/배당 구분)
- FCF = 영업CF - CAPEX
- 기초/기말 현금

## Step 3-B: 주석(Footnotes) 전체 파싱 — 디폴트로 반드시 포함
감사보고서 주석에서 다음을 **모두** 추출한다 (실무에서 주석이 가장 중요):

- **주석1. 주주현황** — 주주명 / 주식수 / 지분율
- **주석4. 투자자산(매도가능증권)** — 자회사·관계사 지분율 및 장부가
- **주석5. 유형자산** — 기초/취득/처분/감가상각/기말 + 취득원가·감가상각누계액
- **주석6. 무형자산** — 항목별(영업권/소프트웨어/브랜드/고객관계 등) 기초~기말 + 상각
- **주석7. 차입금** — 차입처 / 이자율 / 만기 / 담보
- **주석9. 충당부채** — 포인트·환불 등 충당부채 변동
- **주석10. 퇴직급여충당부채** — 변동내역
- **주석11. 자본금** — 발행주식수 / 액면 / 자본금
- **주석12. 주식매수선택권(스톡옵션)** — 부여/행사/소멸 변동
- **주석13. 미처리결손금/이익잉여금**
- **주석14. 특수관계자 거래**
- **주석15. 우발채무·약정사항** — 지급보증, 소송, 보험
- **주석17. 부가가치 관련자료** — 급여·감가상각 등 비용 상세

## Step 4: Excel 생성
**다음 8개 시트를 디폴트로 모두 생성한다** (재무제표 4 + 주석 4):

| Sheet | 내용 |
|---|---|
| ① Summary | 핵심지표 + YoY/QoQ% + 주요비율 |
| ② P&L | 손익계산서 전체 + EBITDA |
| ③ BS | 재무상태표 전체 + 자산 세분화 |
| ④ CF | 현금흐름표 + FCF |
| ⑤ 주석_자산 | 유형자산·무형자산·투자지분 상세 |
| ⑥ 주석_부채자본 | 차입금·충당부채·퇴직급여·자본금·결손금 |
| ⑦ 주석_주주_SO | 주주현황·스톡옵션·특수관계자·우발채무 |
| ⑧ 주석_부가가치 | 부가가치 관련 비용 상세 |

**포맷 규칙:**
- 재무제표 시트 단위: 억원 (KRW), 주석 시트 단위: 천원 (원문 그대로)
- 헤더: 네이비(#1F3864) 배경 / 흰색 텍스트 / 굵게
- 소계·합계 행: 연회색(#F2F2F2) 배경 + 굵게
- YoY/QoQ 변화율: 양수 초록(#00B050) / 음수 빨강(#FF0000)
- 연결재무제표 우선, 없으면 별도재무제표 사용 (시트 상단에 표기)
- 폰트: 맑은 고딕
- 파일명: `{기업명}_재무제표_{시작연도}-{종료연도}_{yoy|qoq}.xlsx`

## Step 5: 채팅 요약 출력
Excel 생성 후 최근 3개년 핵심지표 테이블을 채팅창에도 출력:
- 매출 / 영업이익 / EBITDA / 순이익 / 영업이익률 / 부채비율 / FCF
