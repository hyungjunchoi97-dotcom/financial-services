Load the `dart-financials` skill and execute the following workflow:

## Step 1: 기업 코드 조회
- DART MCP `resolve_corp_code`로 기업명 → corp_code 변환
- 동명이인 기업이 여러 개면 목록 보여주고 선택 요청

## Step 2: 감사보고서 수집
- `search_disclosures`로 해당 기업의 감사보고서(A001) 목록 조회
- 요청 기간 내 연도별 감사보고서 전부 수집
- `download_document`로 각 연도 감사보고서 원문 가져오기
- **비상장사도 외부감사 대상이면 DART에 감사보고서 존재** → 동일하게 처리

## Step 3: 재무제표 파싱
감사보고서에서 다음 항목 추출:

**손익계산서 (P&L)**
- 매출액 / 매출원가 / 매출총이익
- 판매비와관리비 (세부항목: 인건비, 감가상각비, 무형자산상각비 등)
- 영업이익
- 영업외손익 (이자수익, 이자비용, 기타)
- 세전이익 / 법인세 / 당기순이익
- EBITDA = 영업이익 + 감가상각비 + 무형자산상각비

**재무상태표 (BS)**
- 유동자산 (현금, 매출채권, 재고자산, 기타)
- 비유동자산 (유형자산 총계, 무형자산 총계, 투자자산, 기타)
- 유형자산 상세: 취득원가 / 감가상각누계액 / 장부가액
- 무형자산 상세: 취득원가 / 상각누계액 / 장부가액
- 유동부채 / 비유동부채 (차입금 구분)
- 자본금 / 자본잉여금 / 이익잉여금 / 총자본

**현금흐름표 (CF)**
- 영업활동현금흐름
- 투자활동현금흐름 (CAPEX 별도 표시)
- 재무활동현금흐름 (차입/상환/배당 구분)
- FCF = 영업CF - CAPEX
- 기초/기말 현금

## Step 4: Excel 생성
다음 6개 시트로 구성:

| Sheet | 내용 |
|---|---|
| Summary | 핵심지표 + YoY/QoQ% 변화율 |
| P&L | 손익계산서 전체 + EBITDA |
| BS | 재무상태표 전체 + 자산 세분화 |
| CF | 현금흐름표 + FCF |
| PP&E / Intangibles | 유형·무형자산 상세 + 감가상각 |
| Key Ratios | 수익성/안정성/성장성 지표 |

**포맷 규칙:**
- 단위: 억원 (KRW) — 천만원 미만 반올림
- 헤더: 네이비(#1F3864) 배경 / 흰색 텍스트
- YoY/QoQ 변화율: 양수 초록(#00B050) / 음수 빨강(#FF0000)
- 연결재무제표 우선, 없으면 별도재무제표 사용 (시트 상단에 표기)
- 파일명: `{기업명}_감사보고서_{시작연도}-{종료연도}_{yoy|qoq}.xlsx`

## Step 5: 채팅 요약 출력
Excel 생성 후 최근 3개년 핵심지표 테이블을 채팅창에도 출력:
- 매출 / 영업이익 / EBITDA / 순이익 / 영업이익률 / 부채비율 / FCF
